# 湖南高考备考资料 · 整体写作结构

## 项目定位

为湖南省高考（新课标Ⅰ卷）语数外物化生六科分别编写备考资料，每科独立成册（含 PDF），同时提供合订本。每科目标篇幅 **60--100 页**。

---

## 一、统一规范

### 文件结构（每科）

```
学科/
├── README.md                    # 本册说明（试卷结构、分值、使用方法）
├── 学科.tex                     # 独立编译入口（含导言、\input 各章节）
├── formulas.tex                 # ① 常用公式、定理与结论
├── timing-and-tips.tex          # ② 答题时间规划与答题技巧
├── frequency-analysis.tex       # ③ 考频分析
├── guided-practice.tex          # ④ 真题代练
├── categorized-training.tex     # ⑤ 分类训练与限时训练
└── (可选) mock-exams.tex        # ⑥ 综合模拟卷
```

### 格式规范
- 引擎：XeLaTeX（`ctex` 宏包处理中文）
- 编码：UTF-8
- 正文：ctexbook（`\chapter` 级）或 ctexart（`\section` 级）
- 数学公式：`amsmath` + `amssymb`
- 图表：`tabular` / `array`，鼓励用 `booktabs` 美化

### 每节内容要求

| 章节 | 核心内容 | 篇幅占比 |
|------|----------|----------|
| ① 公式定理 | 知识清单、公式表、常用结论 | 15% |
| ② 时间技巧 | 时间分配、答题顺序、各题型技巧 | 10% |
| ③ 考频分析 | 近年考点分布表、命题趋势、复习策略 | 5% |
| ④ 真题代练 | 按知识点精选真题，难题多例题，含完整解析 | 35% |
| ⑤ 分类训练 | 按知识点分类练习题 + 限时模拟卷 | 35% |

## 二、数学 · 详细写作方案

**目标篇幅**：70--80 页

### 章节展开

#### ① `formulas.tex`（已基本完成，约 10 页）
- 12 个模块：集合→复数→不等式→函数→导数→三角→数列→向量→立体几何→解析几何→概率统计→排列组合
- 每模块：核心公式 + 常用结论 + 易错提醒
- 附录：常用数值速查表

#### ② `timing-and-tips.tex`（目标 8--10 页）
- 试卷结构 + 时间分配表
- 答题顺序策略（两种方案）
- 单选题 6 种技巧（直接法/特值法/排除法/数形结合/估算法/选项分析法）+ 例题
- 多选题策略（宁少勿滥 + 排除法实战案例）
- 填空题 5 种技巧（直接计算/数形结合/特殊化/合情推理/正难则反）
- 解答题 6 大题型规范模板 + 踩分点说明
- 检查策略 + 考场应急

#### ③ `frequency-analysis.tex`（目标 5 页）
- 近 5 年考点分值统计表
- 必考/常考/轮考分类
- 题型分布特征
- 命题趋势 + 复习优先级

#### ④ `guided-practice.tex`（当前 7 页 → 目标 25 页）

按知识点逐题编写，每个知识点至少涵盖：
- **基础题** 1--2 道（确保得分）
- **中档题** 2--3 道（真题改编）
- **难题** 2--4 道（真题变式/压轴题）

**重点扩充的题型**（每个至少 3 道完整例题 + 解析）：

| 题型 | 说明 |
|------|------|
| 导数综合 | 含参讨论、隐零点、双变量、极值点偏移、不等式放缩 |
| 解析几何综合 | 定点定值、面积最值、斜率积/和、存在性问题 |
| 立体几何综合 | 不规则图形建系、动点问题、翻折问题 |
| 概率统计综合 | 二项分布与超几何分布辨析、全概率递推、决策问题 |
| 数列综合 | 奇偶项、构造法、不等式放缩与数列求和 |
| 多选题专练 | 函数性质、抽象函数、几何动态判断 |
| 新定义题 | 近年出现的新定义题型示范 |

#### ⑤ `categorized-training.tex`（当前 5 页 → 目标 20 页）

每个知识点分类训练扩充至 8--12 题，含：
- 基础巩固 4 题
- 能力提升 4 题
- 挑战冲刺 2--4 题

限时训练扩充至 5 套：
- 基础卷（30 分钟，目标 40+/44）
- 中档卷（45 分钟，目标 40+/51）
- 拔高卷（60 分钟，目标 30+/39）
- 综合卷（90 分钟，全真模拟）
- 压轴卷（90 分钟，高难度）

#### ⑥ （可选）`mock-exams.tex`（目标 10 页）

1--2 套完整模拟卷（22 题，150 分，120 分钟），标注难度与建议用时。
