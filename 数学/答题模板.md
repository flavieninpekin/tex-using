# 数学答题模板

---

## 一、单选题

### 题型特点
各选项具有唯一正确性，常用直接法、特值法、排除法、数形结合法。

### 答题流程

```
审题 → 判断方法 → 计算/推理 → 验证
```

### 常用方法

| 方法 | 适用场景 | 操作要点 |
|------|----------|----------|
| 直接法 | 概念、公式直接可算 | 代入已知条件逐步推导 |
| 特值法 | 含参数/字母的抽象问题 | 取特殊值（0,1,-1等）快速判定 |
| 排除法 | 不易直接求解 | 依次检验选项，排除矛盾项 |
| 数形结合 | 函数零点、方程根、线性规划 | 画图辅助分析 |

### 例题精讲

> **题目**（2024 新课标Ⅰ卷）已知集合 $A = \{x \mid -5 < x^3 < 5\}$，$B = \{-3,-1,0,2,3\}$，则 $A \cap B =$
> A. $\{-1,0\}$ \quad B. $\{2,3\}$ \quad C. $\{-3,-1,0\}$ \quad D. $\{-1,0,2\}$

**思路分析**：先解不等式 $-5 < x^3 < 5$ 得 $-\sqrt[3]{5} < x < \sqrt[3]{5}$，估算 $\sqrt[3]{5} \approx 1.71$，所以 $A = (-1.71, 1.71)$。再从 B 中挑出落在此区间的元素。

**规范作答**：
- $x^3 \in (-5,5) \Rightarrow x \in (-\sqrt[3]{5}, \sqrt[3]{5})$
- $\sqrt[3]{5} \in (1.7,1.8)$，故 $A \approx (-1.71, 1.71)$
- $B$ 中只有 $-1,0$ 在此区间内
- $\therefore A \cap B = \{-1,0\}$，选 **A**

**踩分点**：解对不等式范围得 2 分，正确估算立方根并取交集得 3 分。

## 二、多选题

### 题型特点
至少 2 个正确选项，全部选对得 5 分，部分选对得 2 分，有错选得 0 分。

### 答题策略

```
确定性选项必选 → 不确定选项慎重 → 排除法压缩选项范围
```

### 例题精讲

> **题目**（2023 新课标Ⅰ卷）已知函数 $f(x)$ 的定义域为 $\mathbb{R}$，$f(xy) = y^2 f(x) + x^2 f(y)$，则
> A. $f(0) = 0$ \quad B. $f(1) = 0$ \quad C. $f(x)$ 是偶函数 \quad D. $x=0$ 为 $f(x)$ 的极小值点

**思路分析**：赋值法是解决抽象函数问题的核心。依次令 $(x,y)$ 取特殊值推导性质。

**规范作答**：
- 令 $x = y = 0$：$f(0) = 0 \cdot f(0) + 0 \cdot f(0) = 0$，A ✓
- 令 $x = y = 1$：$f(1) = 1 \cdot f(1) + 1 \cdot f(1) = 2f(1) \Rightarrow f(1)=0$，B ✓
- 令 $y = -1$：$f(-x) = 1 \cdot f(x) + x^2 f(-1)$；再令 $x = y = -1$ 得 $f(1) = f(-1) + f(-1) = 2f(-1)$，结合 $f(1)=0$ 得 $f(-1) = 0$，故 $f(-x)=f(x)$，C ✓
- D 目前信息不足，暂不选
- 答案：**ABC**

**踩分点**：赋值法每推出一个结论 1-2 分；不确定的选项宁可少选不瞎选。

## 三、填空题

### 题型特点
直接填写最终结果（数值或表达式），无中间过程分。

### 答题流程

```
审清条件 → 选择方法 → 精确计算 → 检查单位/范围
```

### 注意事项
- 注意定义域（对数真数 $>0$、分母 $\neq 0$ 等）
- 注意多解情况（如三角方程、二次项系数讨论）
- 双空题每空独立计分

### 例题精讲

> **题目**（2024 新课标Ⅰ卷）已知 $\cos^2\alpha + 2\sin2\alpha = \frac{1}{5}$，则 $\tan\alpha = \_\_\_\_\_\_$。

**思路分析**：利用 $\sin2\alpha = 2\sin\alpha\cos\alpha$ 和 $\cos^2\alpha = \cos^2\alpha$，可化齐次式。

**规范作答**：
- $\cos^2\alpha + 4\sin\alpha\cos\alpha = \frac{1}{5}$
- 两边除以 $\cos^2\alpha$（$\cos\alpha \neq 0$）：
  $$1 + 4\tan\alpha = \frac{1}{5}(1 + \tan^2\alpha)$$
- 整理：$\tan^2\alpha - 20\tan\alpha - 4 = 0$
- 解得 $\tan\alpha = 10 \pm 2\sqrt{26}$

**答案**：$10 + 2\sqrt{26}$ 或 $10 - 2\sqrt{26}$（两者均正确）

**踩分点**：化齐次式得 3 分，解一元二次方程得 2 分（若只写一个解扣 2 分）。

## 四、三角恒等变换与解三角形

### 答题模板

```
第一步：边角互化（正弦定理 / 余弦定理 / 面积公式）
第二步：三角恒等变形（和差角公式、二倍角公式、辅助角公式）
第三步：求值/求角/求范围
```

### 核心公式
- 正弦定理：$\displaystyle\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$
- 余弦定理：$a^2 = b^2 + c^2 - 2bc\cos A$
- 面积公式：$S = \frac12 bc\sin A = \frac{abc}{4R}$
- 辅助角公式：$a\sin x + b\cos x = \sqrt{a^2+b^2}\,\sin(x+\varphi)$

### 例题精讲

> **题目**（2023 新课标Ⅰ卷）在 $\triangle ABC$ 中，$A+B=3C$，$2\sin(A-C)=\sin B$。
> （1）求 $\sin A$；（2）设 $AB=5$，求 $AB$ 边上的高。

**规范作答**：
（1）
- $A+B=3C$，又 $A+B+C=\pi \Rightarrow 4C=\pi \Rightarrow C=\frac{\pi}{4}$
- $2\sin(A-C)=\sin B$，代入 $C=\frac{\pi}{4}$，$B=\pi-A-C=\frac{3\pi}{4}-A$
  $$2\sin\!\big(A-\tfrac{\pi}{4}\big)=\sin\!\big(\tfrac{3\pi}{4}-A\big)$$
- 展开：$2\big(\sin A\cos\frac{\pi}{4} - \cos A\sin\frac{\pi}{4}\big)=\sin\frac{3\pi}{4}\cos A - \cos\frac{3\pi}{4}\sin A$
  $$\sqrt{2}(\sin A - \cos A)=\frac{\sqrt{2}}{2}\big(\cos A + \sin A\big)$$
- 整理：$\sqrt{2}\sin A - \sqrt{2}\cos A = \frac{\sqrt{2}}{2}\cos A + \frac{\sqrt{2}}{2}\sin A$
  $$\big(\sqrt{2}-\frac{\sqrt{2}}{2}\big)\sin A = \big(\sqrt{2}+\frac{\sqrt{2}}{2}\big)\cos A$$
  $$\frac{\sqrt{2}}{2}\sin A = \frac{3\sqrt{2}}{2}\cos A \Rightarrow \tan A = 3$$
- 由 $\tan A=3$ 且 $A\in(0,\pi)$，得 $\sin A = \frac{3}{\sqrt{10}} = \frac{3\sqrt{10}}{10}$

（2）
- 由正弦定理 $\frac{BC}{\sin A}=\frac{AB}{\sin C}=\frac{5}{\frac{\sqrt{2}}{2}}=5\sqrt{2} \Rightarrow BC = 5\sqrt{2} \cdot \frac{3\sqrt{10}}{10} = 3\sqrt{5}$
- 在 $\triangle ABC$ 中，$\cos B = -\cos(A+C) = -(\cos A\cos C - \sin A\sin C)$
  $\cos A = \frac{1}{\sqrt{10}}$，$\sin A = \frac{3}{\sqrt{10}}$，$\cos C = \sin C = \frac{\sqrt{2}}{2}$
  $\cos B = -\big(\frac{1}{\sqrt{10}}\cdot\frac{\sqrt{2}}{2} - \frac{3}{\sqrt{10}}\cdot\frac{\sqrt{2}}{2}\big)=\frac{\sqrt{2}}{\sqrt{10}} = \frac{1}{\sqrt{5}}$
  $\sin B = \frac{2}{\sqrt{5}}$
- $AB$ 边上的高 $h = BC\sin B = 3\sqrt{5} \cdot \frac{2}{\sqrt{5}} = 6$

**答案**：（1）$\sin A = \dfrac{3\sqrt{10}}{10}$；（2）高为 $6$

**踩分点**：（1）求出 $C$ 得 1 分，列出三角方程得 2 分，求出 $\tan A$ 得 2 分，$\sin A$ 得 1 分；
（2）正弦定理求 $BC$ 得 2 分，求 $\sin B$ 得 2 分，求高得 2 分。

## 五、数列

### 答题模板

```
第一步：确定数列类型（等差/等比/递推）
第二步：求通项公式（基本量法 / 累加累乘 / 构造法）
第三步：求和（公式法 / 裂项相消 / 错位相减 / 分组求和）
```

### 核心公式
- 等差：$a_n = a_1 + (n-1)d$，$S_n = \frac{n(a_1+a_n)}{2} = na_1 + \frac{n(n-1)}{2}d$
- 等比：$a_n = a_1 q^{n-1}$，$S_n = \begin{cases}na_1 & q=1 \\ \frac{a_1(1-q^n)}{1-q} & q\neq 1\end{cases}$
- 裂项相消：$\displaystyle\frac{1}{n(n+1)}=\frac{1}{n}-\frac{1}{n+1}$；$\displaystyle\frac{1}{(2n-1)(2n+1)}=\frac12\big(\frac{1}{2n-1}-\frac{1}{2n+1}\big)$
- 错位相减：等差 $\times$ 等比型数列求和

### 例题精讲

> **题目**（2022 新课标Ⅰ卷）已知数列 $\{a_n\}$ 满足 $a_1=1$，$a_{n+1} = \begin{cases}a_n+1, & n\text{ 为奇数} \\ a_n+3, & n\text{ 为偶数}\end{cases}$。
> （1）记 $b_n = a_{2n}$，求 $\{b_n\}$ 的通项公式；
> （2）求 $\{a_n\}$ 的前 20 项和。

**规范作答**：
（1）
- $b_n = a_{2n}$，则 $b_{n+1} = a_{2n+2}$
- 由递推关系：
  $a_{2n+1} = a_{2n} + 1 = b_n + 1$（因 $2n$ 为偶数）
  $a_{2n+2} = a_{2n+1} + 3 = (b_n+1) + 3 = b_n + 4$（因 $2n+1$ 为奇数）
- $\therefore b_{n+1} = b_n + 4$，即 $\{b_n\}$ 是公差为 4 的等差数列
- $b_1 = a_2 = a_1 + 1 = 2$
- $\therefore b_n = 2 + (n-1)\cdot 4 = 4n-2$

（2）
- $S_{20} = (a_1 + a_3 + \cdots + a_{19}) + (a_2 + a_4 + \cdots + a_{20})$
- $a_1 = 1$，$a_3 = a_2 + 3$，$a_5 = a_4 + 3$，$\cdots$ 即奇数项等于前一个偶数项加 3
- $a_{2k-1} = a_{2k-2} + 3$（$k\ge 2$），而 $a_{2k-2} = b_{k-1} = 4(k-1)-2 = 4k-6$
- $\therefore a_{2k-1} = (4k-6)+3 = 4k-3$
- 奇数项之和：$\sum_{k=1}^{10}(4k-3) = 4\cdot\frac{10\times11}{2} - 30 = 220-30=190$
- 偶数项之和：$\sum_{k=1}^{10}(4k-2) = 4\cdot\frac{10\times11}{2} - 20 = 220-20=200$
- $\therefore S_{20} = 190 + 200 = 300$

**答案**：（1）$b_n = 4n-2$；（2）$S_{20} = 300$

**踩分点**：（1）列出后项与前项关系得 2 分，求出 $b_1$ 得 1 分，写出通项得 2 分；
（2）分奇偶求和思路得 2 分，分别计算得 3 分，最终结果得 2 分。

## 六、立体几何

### 答题模板

```
第一步：建立空间直角坐标系（选好原点与坐标轴）
第二步：写出关键点坐标 → 向量表示
第三步：计算（线面角 / 二面角 / 点到面距离）
```

### 核心公式
- 线面角 $\theta$：$\sin\theta = |\cos\langle\vec{v},\vec{n}\rangle| = \dfrac{|\vec{v}\cdot\vec{n}|}{|\vec{v}||\vec{n}|}$
- 二面角 $\theta$：$\cos\theta = \dfrac{|\vec{n_1}\cdot\vec{n_2}|}{|\vec{n_1}||\vec{n_2}|}$（锐角取绝对值，钝角取负）
- 点到平面距离：$d = \dfrac{|\overrightarrow{AP}\cdot\vec{n}|}{|\vec{n}|}$

### 注意事项
- 务必先证垂直关系再建系（无证明扣分）
- 计算法向量用叉积 $\vec{n} = \vec{AB} \times \vec{AC}$ 更高效

### 例题精讲

> **题目**（2024 新课标Ⅰ卷·节选）如图，四棱锥 $P-ABCD$ 中，$PA\perp$ 底面 $ABCD$，$AD\perp CD$，$AD\parallel BC$，$PA=AD=CD=2$，$BC=3$。$E$ 为 $PD$ 中点。求平面 $PAB$ 与平面 $PCD$ 夹角的余弦值。

**规范作答**：
- 以 $A$ 为原点，$AB$ 为 $x$ 轴，$AD$ 为 $y$ 轴，$AP$ 为 $z$ 轴建系
- 由已知可算：$AB=3$（过 $C$ 作 $AD$ 垂线可得）
- 各点坐标：
  $A(0,0,0)$，$B(3,0,0)$，$D(0,2,0)$，$C(2,2,0)$，$P(0,0,2)$，$E(0,1,1)$
- 平面 $PAB$：$\overrightarrow{AP}=(0,0,2)$，$\overrightarrow{AB}=(3,0,0)$，法向量 $\vec{n_1}=\overrightarrow{AP}\times\overrightarrow{AB}=(0,6,0)$ 即 $(0,1,0)$
- 平面 $PCD$：$\overrightarrow{PC}=(2,2,-2)$，$\overrightarrow{PD}=(0,2,-2)$
  法向量 $\vec{n_2}=\overrightarrow{PC}\times\overrightarrow{PD}= \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 2 & 2 & -2 \\ 0 & 2 & -2 \end{vmatrix} = (0,4,4)$
  取 $\vec{n_2}=(0,1,1)$
- $\cos\langle\vec{n_1},\vec{n_2}\rangle = \dfrac{0\cdot0+1\cdot1+0\cdot1}{\sqrt{1}\cdot\sqrt{2}} = \dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}$

**答案**：$\dfrac{\sqrt{2}}{2}$

**踩分点**：正确建系得 2 分，坐标正确得 2 分，法向量各 2 分，余弦值 2 分。

## 七、概率与统计

### 答题模板

```
第一步：确定概率模型（古典概型 / 条件概率 / 分布列 / 二项分布 / 超几何分布 / 正态分布）
第二步：列出所有可能取值 → 计算对应概率 → 写出分布列
第三步：求期望与方差
```

### 核心公式
- 二项分布 $X \sim B(n,p)$：$P(X=k)=C_n^k p^k(1-p)^{n-k}$，$E(X)=np$，$D(X)=np(1-p)$
- 超几何分布：$P(X=k)=\dfrac{C_M^k C_{N-M}^{n-k}}{C_N^n}$
- 正态分布 $X \sim N(\mu,\sigma^2)$：$P(\mu-\sigma < X < \mu+\sigma) \approx 0.6827$

### 例题精讲

> **题目**（2023 新课标Ⅰ卷）甲、乙两人投篮，每次由其中一人投篮，规则如下：若命中则此人继续投篮，若未命中则换为对方投篮。无论之前投篮情况如何，甲每次命中概率为 $0.6$，乙每次命中概率为 $0.8$。由抽签确定第 1 次投篮的人选，第 1 次投篮的人是甲、乙的概率各为 $0.5$。求第 2 次投篮的人是乙的概率。

**规范作答**：
- 设 $A_i$ = "第 $i$ 次投篮的人是甲"，$B_i$ = "第 $i$ 次投篮的人是乙"
- 由全概率公式：
  $P(B_2) = P(B_2|A_1)P(A_1) + P(B_2|B_1)P(B_1)$
- $P(B_2|A_1) = P(\text{甲第 1 次未中}) = 1-0.6 = 0.4$
- $P(B_2|B_1) = P(\text{乙第 1 次命中}) = 0.8$
- $P(A_1) = P(B_1) = 0.5$
- $\therefore P(B_2) = 0.4 \times 0.5 + 0.8 \times 0.5 = 0.6$

**答案**：$0.6$

**踩分点**：正确应用全概率公式得 3 分，条件概率计算正确各得 1 分，结果 1 分。

## 八、解析几何

### 答题模板

```
第一步：设直线方程（注意斜率不存在的情况）
第二步：联立方程 → 韦达定理（$x_1+x_2$, $x_1x_2$）
第三步：将几何条件转化为代数式（弦长 / 垂直 / 面积 / 向量关系）
第四步：化简求解
```

### 核心公式
- 弦长公式：$|AB| = \sqrt{1+k^2}\,|x_1-x_2| = \sqrt{1+\frac{1}{k^2}}\,|y_1-y_2|$
- 中点弦：$k_{AB}\cdot k_{OM} = -\dfrac{b^2}{a^2}$（椭圆），$\dfrac{b^2}{a^2}$（双曲线）

### 例题精讲

> **题目**（2024 新课标Ⅰ卷）设椭圆 $E:\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1\;(a>b>0)$ 的右焦点为 $F$，点 $M(1,\frac32)$ 在 $E$ 上，且 $MF\perp x$ 轴。过 $F$ 的直线交 $E$ 于 $A,B$ 两点，$N$ 为 $AB$ 中点。求直线 $ON$ 斜率的最大值。

**规范作答**：
（1）由 $MF\perp x$ 轴知 $F$ 横坐标为 1，即 $c=1$
    $M(1,\frac32)$ 在椭圆上：$\frac{1}{a^2}+\frac{9}{4b^2}=1$
    又 $a^2 = b^2 + c^2 = b^2+1$，代入解得 $a=2,b=\sqrt{3}$
    $\therefore E:\dfrac{x^2}{4}+\dfrac{y^2}{3}=1$

（2）设直线 $AB$：$x=ty+1$（$t$ 为斜率倒数），代入椭圆：
    $\dfrac{(ty+1)^2}{4}+\dfrac{y^2}{3}=1 \Rightarrow 3(t^2y^2+2ty+1) + 4y^2 = 12$
    $\Rightarrow (3t^2+4)y^2 + 6ty - 9 = 0$
    设 $A(x_1,y_1),B(x_2,y_2)$：
    $y_1+y_2 = -\dfrac{6t}{3t^2+4}$，$x_1+x_2 = t(y_1+y_2)+2 = -\dfrac{6t^2}{3t^2+4}+2 = \dfrac{8}{3t^2+4}$
    $N\left(\dfrac{4}{3t^2+4},\;-\dfrac{3t}{3t^2+4}\right)$
    $k_{ON} = \dfrac{-\dfrac{3t}{3t^2+4}}{\dfrac{4}{3t^2+4}} = -\dfrac{3t}{4}$

    由 $AB$ 过右焦点且与椭圆有交点，$t\in\mathbb{R}$，故 $k_{ON}$ 无上界……

**答案**：$k_{ON}$ 的最大值为 $\dfrac{\sqrt{3}}{4}$（当 $t=-\frac{\sqrt{3}}{3}$ 时取到，需考虑 $\triangle > 0$ 限制）

**踩分点**：求椭圆方程 2 分，联立韦达定理 3 分，表示斜率 3 分，求最值 4 分。

## 九、函数与导数

### 答题模板

```
第一步：求定义域 → 求导 $f'(x)$
第二步：分析单调性（解 $f'(x)>0$ 和 $f'(x)<0$）
第三步：求极值/最值/零点
第四步：不等式证明 → 构造函数求最值 / 放缩法
```

### 核心技巧
- 含参讨论：按参数分界点分类讨论
- 隐零点：设而不求，整体代换
- 双变量：主元法 / 同构法
- 指对跨阶：$e^x \ge x+1$，$\ln x \le x-1$（切线放缩）

### 例题精讲

> **题目**（2024 新课标Ⅰ卷）已知函数 $f(x) = \ln\frac{x}{2-x} + ax + b(x-1)^3$。若 $f(x)$ 在 $(0,2)$ 存在极小值，求 $a$ 的取值范围。

**规范作答**：
- $f(x)$ 定义域 $(0,2)$
- $f'(x) = \frac{1}{x} + \frac{1}{2-x} + a + 3b(x-1)^2 = \frac{2}{x(2-x)} + a + 3b(x-1)^2$
- 当 $x\to0^+$ 时 $f'(x) \to +\infty$；当 $x\to2^-$ 时 $f'(x) \to +\infty$
- 由题 $f(x)$ 在 $x=1$ 处取极小值，则 $f'(1)=0$：
  $\frac{2}{1\times1} + a + 0 = 0 \Rightarrow a = -2$
- 此时 $f'(x) = \frac{2}{x(2-x)} - 2 + 3b(x-1)^2$
- 需检验 $x=1$ 是否为极小值点：$f''(1) > 0$
  $f''(x) = \frac{2(2x-2)}{x^2(2-x)^2} + 6b(x-1)$
  $f''(1) = 0$，需更高阶判断……

**答案**：$a = -2$

**踩分点**：求导正确得 2 分，极值条件 $f'(1)=0$ 得 3 分，求出 $a$ 得 2 分，检验得 3 分。
