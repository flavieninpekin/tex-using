## 一、集合与常用逻辑用语

| 内容 | 公式 / 结论 |
|------|------------|
| 集合运算 | $A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$ |
|  | $A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$ |
|  | $\complement_U A = \{x \mid x \in U \text{ 且 } x \notin A\}$ |
| 摩根律 | $\complement_U(A \cap B) = \complement_U A \cup \complement_U B$ |
|  | $\complement_U(A \cup B) = \complement_U A \cap \complement_U B$ |
| 子集个数 | 含 $n$ 个元素的集合有 $2^n$ 个子集，$2^n-1$ 个真子集 |
| 充分必要条件 | $p \Rightarrow q$：充分；$q \Rightarrow p$：必要；互推：充要 |
| 量词否定 | $\forall x, p(x)$ 否定为 $\exists x, \lnot p(x)$ |

## 二、复数

| 内容 | 公式 |
|------|------|
| 代数形式 | $z = a + bi$，$i^2 = -1$ |
| 共轭复数 | $\bar{z} = a - bi$ |
| 模 | $|z| = \sqrt{a^2 + b^2}$ |
| 四则运算 | $(a+bi) \pm (c+di) = (a\pm c) + (b\pm d)i$ |
|  | $(a+bi)(c+di) = (ac-bd) + (ad+bc)i$ |
|  | $\dfrac{a+bi}{c+di} = \dfrac{(a+bi)(c-di)}{c^2+d^2}$ |
| 几何意义 | 复平面点 $(a,b)$，模为到原点距离 |
| 实系数虚根 | 实系数方程虚根成对出现（共轭） |

## 三、不等式

| 内容 | 公式 / 结论 |
|------|------------|
| 基本不等式 | $\dfrac{a+b}{2} \ge \sqrt{ab}$（$a,b > 0$），当 $a=b$ 取等 |
|  | $a^2 + b^2 \ge 2ab$，当 $a=b$ 取等 |
| 重要不等式 | $a^2 + b^2 + c^2 \ge ab + bc + ca$ |
|  | $\dfrac{b}{a} + \dfrac{a}{b} \ge 2$（$ab>0$） |
| 一元二次不等式 | $ax^2+bx+c > 0$ 解法：先看 $a$ 正负，再求根，再画图 |
| 绝对值不等式 | $|a|-|b| \le |a\pm b| \le |a|+|b|$ |
| 柯西不等式 | $(a^2+b^2)(c^2+d^2) \ge (ac+bd)^2$ |
| 线性规划 | 先画可行域，平移目标函数找最值 |

## 四、函数与导数

### 基本初等函数

| 函数 | 定义域 | 性质要点 |
|------|--------|----------|
| $y=x^\alpha$ | 视 $\alpha$ 而定 | 过 $(1,1)$；$\alpha>0$ 时过 $(0,0)$ |
| $y=a^x$ | $\mathbb{R}$ | $a>1$ 增，$0<a<1$ 减；过 $(0,1)$；值域 $(0,+\infty)$ |
| $y=\log_a x$ | $(0,+\infty)$ | $a>1$ 增，$0<a<1$ 减；过 $(1,0)$；值域 $\mathbb{R}$ |
| $y=\sin x$ | $\mathbb{R}$ | 奇函数，周期 $2\pi$，值域 $[-1,1]$ |
| $y=\cos x$ | $\mathbb{R}$ | 偶函数，周期 $2\pi$，值域 $[-1,1]$ |
| $y=\tan x$ | $x\neq\frac{\pi}{2}+k\pi$ | 奇函数，周期 $\pi$，值域 $\mathbb{R}$ |

### 函数的性质

| 性质 | 判定方法 |
|------|----------|
| 奇偶性 | $f(-x)=f(x)$ 偶函数（关于 $y$ 轴对称） |
|  | $f(-x)=-f(x)$ 奇函数（关于原点对称） |
| 单调性 | $f'(x) \ge 0$ 增，$f'(x) \le 0$ 减 |
| 周期性 | $f(x+T)=f(x)$，最小正周期为 $T$ |
| 对称性 | $f(a+x)=f(a-x)$ $\Leftrightarrow$ 关于 $x=a$ 对称 |
|  | $f(a+x)+f(a-x)=2b$ $\Leftrightarrow$ 关于 $(a,b)$ 对称 |

### 导数公式

| $f(x)$ | $f'(x)$ |
|--------|---------|
| $C$（常数） | $0$ |
| $x^n$ | $nx^{n-1}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\tan x$ | $\sec^2 x$ |
| $a^x$ | $a^x \ln a$ |
| $e^x$ | $e^x$ |
| $\log_a x$ | $\dfrac{1}{x\ln a}$ |
| $\ln x$ | $\dfrac{1}{x}$ |

### 导数运算法则

| 法则 | 公式 |
|------|------|
| 和差 | $(u \pm v)' = u' \pm v'$ |
| 积 | $(uv)' = u'v + uv'$ |
| 商 | $\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2}$ |
| 链式 | $f(g(x))' = f'(g(x)) \cdot g'(x)$ |

### 常用切线放缩

| 不等式 | 取等条件 |
|--------|----------|
| $e^x \ge x+1$ | $x=0$ |
| $e^x \ge ex$ | $x=1$ |
| $\ln x \le x-1$ | $x=1$ |
| $\ln x \ge 1 - \dfrac{1}{x}$ | $x=1$ |
| $\sin x \le x \le \tan x$ | $x=0$（$x\in[0,\frac{\pi}{2})$） |

## 五、三角函数与解三角形

### 三角恒等变换

| 公式 | 表达式 |
|------|--------|
| 同角关系 | $\sin^2\alpha + \cos^2\alpha = 1$，$\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha}$ |
| 和差角 | $\sin(\alpha\pm\beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$ |
|  | $\cos(\alpha\pm\beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$ |
|  | $\tan(\alpha\pm\beta) = \dfrac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$ |
| 二倍角 | $\sin2\alpha = 2\sin\alpha\cos\alpha$ |
|  | $\cos2\alpha = \cos^2\alpha-\sin^2\alpha = 2\cos^2\alpha-1 = 1-2\sin^2\alpha$ |
|  | $\tan2\alpha = \dfrac{2\tan\alpha}{1-\tan^2\alpha}$ |
| 降幂 | $\sin^2\alpha = \dfrac{1-\cos2\alpha}{2}$，$\cos^2\alpha = \dfrac{1+\cos2\alpha}{2}$ |
| 辅助角 | $a\sin x + b\cos x = \sqrt{a^2+b^2}\,\sin(x+\varphi)$，$\tan\varphi = \dfrac{b}{a}$ |
| 诱导 | 奇变偶不变，符号看象限 |

### 解三角形

| 公式 | 表达式 |
|------|--------|
| 正弦定理 | $\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$ |
| 余弦定理 | $a^2 = b^2 + c^2 - 2bc\cos A$ |
| 面积公式 | $S = \dfrac12 bc\sin A = \dfrac12 ac\sin B = \dfrac12 ab\sin C$ |
|  | $S = \dfrac{abc}{4R} = \dfrac12 (a+b+c)r$ |

## 六、数列

| 类型 | 通项公式 | 前 $n$ 项和 |
|------|----------|-------------|
| 等差数列 | $a_n = a_1 + (n-1)d$ | $S_n = \dfrac{n(a_1+a_n)}{2} = na_1 + \dfrac{n(n-1)}{2}d$ |
| 等比数列 | $a_n = a_1 q^{n-1}$ | $S_n = \begin{cases}na_1 & q=1 \\ \dfrac{a_1(1-q^n)}{1-q} & q\neq 1\end{cases}$ |

### 常用求和方法

| 方法 | 适用类型 | 示例 |
|------|----------|------|
| 公式法 | 等差/等比 | 直接套公式 |
| 裂项相消 | 分式型 | $\dfrac{1}{n(n+1)} = \dfrac{1}{n} - \dfrac{1}{n+1}$ |
|  |  | $\dfrac{1}{(2n-1)(2n+1)} = \dfrac12\big(\dfrac{1}{2n-1} - \dfrac{1}{2n+1}\big)$ |
|  |  | $\dfrac{1}{\sqrt{n}+\sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$ |
| 错位相减 | 等差 $\times$ 等比 | $S_n = \sum (an+b)q^{n-1}$ |
| 分组求和 | 两组不同数列 | 分别求和再相加 |
| 并项求和 | 正负交替 | $(-1)^n n$ 型，按 $n$ 奇偶讨论 |

## 七、平面向量

| 内容 | 公式 |
|------|------|
| 坐标表示 | $\vec{a} = (x,y)$ |
| 模 | $|\vec{a}| = \sqrt{x^2+y^2}$ |
| 数量积 | $\vec{a}\cdot\vec{b} = x_1x_2 + y_1y_2 = |\vec{a}||\vec{b}|\cos\theta$ |
| 垂直 | $\vec{a} \perp \vec{b} \Leftrightarrow \vec{a}\cdot\vec{b} = 0 \Leftrightarrow x_1x_2+y_1y_2=0$ |
| 平行 | $\vec{a} \parallel \vec{b} \Leftrightarrow \vec{a}=\lambda\vec{b} \Leftrightarrow x_1y_2 = x_2y_1$ |
| 夹角 | $\cos\theta = \dfrac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|}$ |
| 投影 | $\vec{a}$ 在 $\vec{b}$ 上的投影：$\dfrac{\vec{a}\cdot\vec{b}}{|\vec{b}|}$ |
| 三角形法则 | $\overrightarrow{AB} + \overrightarrow{BC} = \overrightarrow{AC}$ |
| 中点公式 | $M$ 为 $AB$ 中点 $\Leftrightarrow \overrightarrow{OM} = \dfrac12(\overrightarrow{OA} + \overrightarrow{OB})$ |
| 重心 | $G$ 为 $\triangle ABC$ 重心 $\Leftrightarrow \overrightarrow{GA} + \overrightarrow{GB} + \overrightarrow{GC} = \vec{0}$ |

## 八、立体几何

### 空间向量

| 内容 | 公式 |
|------|------|
| 点积 | $\vec{a}\cdot\vec{b} = x_1x_2+y_1y_2+z_1z_2$ |
| 叉积 | $\vec{a}\times\vec{b} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ x_1 & y_1 & z_1 \\ x_2 & y_2 & z_2 \end{vmatrix}$ |
| 法向量 | $\vec{n} = \vec{AB} \times \vec{AC}$ |
| 线面角 | $\sin\theta = \dfrac{|\vec{v}\cdot\vec{n}|}{|\vec{v}||\vec{n}|}$ |
| 二面角 | $\cos\theta = \dfrac{|\vec{n_1}\cdot\vec{n_2}|}{|\vec{n_1}||\vec{n_2}|}$ 或 $\dfrac{\vec{n_1}\cdot\vec{n_2}}{|\vec{n_1}||\vec{n_2}|}$（看锐钝） |
| 点到面距离 | $d = \dfrac{|\overrightarrow{AP}\cdot\vec{n}|}{|\vec{n}|}$ |
| 异面直线距离 | $d = \dfrac{|(\vec{a}\times\vec{b})\cdot\overrightarrow{AB}|}{|\vec{a}\times\vec{b}|}$ |

| 台体 | $V = \frac13(S'+\sqrt{S'S}+S)h$ | — |
| 球体 | $V = \frac43\pi R^3$ | $S = 4\pi R^2$ |

## 九、平面解析几何

### 直线

| 内容 | 公式 |
|------|------|
| 斜率 | $k = \dfrac{y_2-y_1}{x_2-x_1}$ |
| 点斜式 | $y-y_1 = k(x-x_1)$ |
| 两点式 | $\dfrac{y-y_1}{y_2-y_1} = \dfrac{x-x_1}{x_2-x_1}$ |
| 距离公式 | $d = \dfrac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}$（点到直线） |
|  | $d = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$（两点间） |
| 平行线距离 | $d = \dfrac{|C_1-C_2|}{\sqrt{A^2+B^2}}$ |

### 圆

| 内容 | 公式 |
|------|------|
| 标准方程 | $(x-a)^2 + (y-b)^2 = r^2$ |
| 一般方程 | $x^2+y^2+Dx+Ey+F=0$，圆心 $(-\frac{D}{2},-\frac{E}{2})$ |
| 切线 | $x_0x+y_0y = r^2$（圆上点切线） |

### 圆锥曲线

| 曲线 | 标准方程 | $a,b,c$ 关系 | 离心率 $e$ | 渐近线 |
|------|----------|-------------|-----------|--------|
| 椭圆 | $\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1$ | $c^2 = a^2-b^2$ | $e=\dfrac{c}{a} \in (0,1)$ | — |
| 双曲线 | $\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}=1$ | $c^2 = a^2+b^2$ | $e=\dfrac{c}{a} > 1$ | $y=\pm\dfrac{b}{a}x$ |
| 抛物线 | $y^2 = 2px$ | $p$ 为焦准距 | $e=1$ | — |

### 弦长与中点弦

| 内容 | 公式 |
|------|------|
| 弦长 | $|AB| = \sqrt{1+k^2}\,|x_1-x_2| = \sqrt{1+\frac{1}{k^2}}\,|y_1-y_2|$ |
| 椭圆中点弦 | $k_{AB} \cdot k_{OM} = -\dfrac{b^2}{a^2}$ |
| 双曲线中点弦 | $k_{AB} \cdot k_{OM} = \dfrac{b^2}{a^2}$ |
| 抛物线中点弦 | $k_{AB} = \dfrac{p}{y_0}$（$y_0$ 为中点纵坐标） |

## 十、概率与统计

### 概率

| 内容 | 公式 |
|------|------|
| 古典概型 | $P(A) = \dfrac{n(A)}{n(\Omega)}$ |
| 条件概率 | $P(B\mid A) = \dfrac{P(AB)}{P(A)}$ |
| 全概率 | $P(B) = \sum_{i=1}^n P(A_i)P(B\mid A_i)$ |
| 贝叶斯 | $P(A_i\mid B) = \dfrac{P(A_i)P(B\mid A_i)}{\sum_{j=1}^n P(A_j)P(B\mid A_j)}$ |
| 独立事件 | $P(AB) = P(A)P(B)$ |

### 分布列

| 分布 | 概率 | 期望 | 方差 |
|------|------|------|------|
| 二项分布 $B(n,p)$ | $P(X=k)=C_n^k p^k(1-p)^{n-k}$ | $np$ | $np(1-p)$ |
| 超几何分布 | $P(X=k)=\dfrac{C_M^k C_{N-M}^{n-k}}{C_N^n}$ | $n\frac{M}{N}$ | — |
| 两点分布 | $P(X=1)=p$ | $p$ | $p(1-p)$ |

### 统计

| 内容 | 公式 |
|------|------|
| 均值 | $\bar{x} = \dfrac{1}{n}\sum_{i=1}^n x_i$ |
| 方差 | $s^2 = \dfrac{1}{n}\sum_{i=1}^n (x_i-\bar{x})^2 = \dfrac{1}{n}\sum x_i^2 - \bar{x}^2$ |
| 标准差 | $s = \sqrt{s^2}$ |
| 相关系数 | $r = \dfrac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum(x_i-\bar{x})^2\sum(y_i-\bar{y})^2}}$ |
| 回归方程 | $\hat{y} = \hat{b}x + \hat{a}$，$\hat{b} = \dfrac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sum(x_i-\bar{x})^2}$ |
| 正态分布 | $X \sim N(\mu,\sigma^2)$：$P(\mu-\sigma<X<\mu+\sigma)\approx 0.6827$ |
|  | $P(\mu-2\sigma<X<\mu+2\sigma)\approx 0.9545$ |
|  | $P(\mu-3\sigma<X<\mu+3\sigma)\approx 0.9973$ |

## 十一、排列组合与二项式定理

| 内容 | 公式 |
|------|------|
| 排列 | $A_n^m = \dfrac{n!}{(n-m)!} = n(n-1)\cdots(n-m+1)$ |
| 组合 | $C_n^m = \dfrac{n!}{m!(n-m)!} = \dfrac{A_n^m}{m!}$ |
| 组合性质 | $C_n^m = C_n^{n-m}$，$C_n^m + C_n^{m+1} = C_{n+1}^{m+1}$ |
| 二项展开 | $(a+b)^n = \sum_{k=0}^n C_n^k a^{n-k}b^k$ |
| 通项 | $T_{k+1} = C_n^k a^{n-k}b^k$ |
| 二项式系数和 | $C_n^0 + C_n^1 + \cdots + C_n^n = 2^n$ |
| 奇数项系数 | $C_n^0 + C_n^2 + C_n^4 + \cdots = 2^{n-1}$ |
